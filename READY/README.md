## 🚀 Как использовать:

### Вариант 1: На компе (локально)

```bash
# 1. Установи Node.js: https://nodejs.org

# 2. Скачай эту папку и открой терминал в ней

# 3. Установи зависимости:
npm install

# 4. Запусти:
npm run dev

# 5. Открой в браузере:
http://localhost:3000
```

Готово! 🎉

---

### Вариант 2: На Vercel (облако - РЕКОМЕНДУЕТСЯ)

**Это вообще супер просто:**

1. Зайди: https://github.com/new
2. Создай новый репо `good-news-final`
3. Загрузи все файлы из этой папки
4. Зайди: https://vercel.com
5. Нажми "Import Project"
6. Выбери свой репо
7. Нажми "Deploy"
8. **ГОТОВО! Сайт в интернете!** ☁️
